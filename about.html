OUR STORY

把禱告變成可被看見、被回應的行動

Start Pray 由一群熱衷禱告與科技的夥伴所組成，期待運用區塊鏈的透明與信任特性，凝聚世界各地的守望者。我們相信每一個禱告都值得被紀錄、被回應，也值得在社群中看見真實的陪伴。

  • 建立安全的禱告發佈與回應流程，避免信息被遺忘
  • 透過 代幣鼓勵持續的陪伴與語音禱告
  • 提供教會、機構、個人都能使用的開放生態

    • 我們的核心價值

    從禱告到行動，我們以四個關鍵價值推動 Start Pray 的產品發展與社群文化。

    真實陪伴

    結合文字與語音禱告，讓彼此的生命故事被聽見，取代快速、片段的社群互動。

    透明信任

    以區塊鏈紀錄代幣發放與消耗，打造可追蹤、可驗證的守望流程。

    共享資源

    提供 API 與 SDK，讓教會、非營利組織與開發者能快速串接，延伸祈禱網絡。

    合規守護

    遵循區域法規、重視資料隱私，確保每筆禱告、每次代幣交易都在安全環境中進行。

    MISSION

    從本地小組到全球祈禱網絡

    Start Pray 的雛形來自於一個線上小組。疫情期間，我們經歷了遠距禱告的限制，因而開始思考：如何讓禱告不因距離而消失？如何記錄每一次回應的恩典？

    透過數位工具與代幣激勵，我們看見跨文化、跨城市的守望者相互扶持，也看見禱告所帶來的真實改變。

    • Beta 測試涵蓋 12 個國家、60+ 禱告小組
    • 導入語音轉寫與自動代幣回饋機制
    • 提供儀表板即時洞察祈禱影響力
